Estimate Home Price

This website gathers data from 21,613 home sales in King County, Washington from the years 2014 and 2015. The data is then used to create a machine learning model using Random Forest Regression. You can use this model to estimate any home's price. Based on the values you input here for the various home characteristics, the model will return a price based on those features. Enter a value for each of the 9 characteristics below and click 'Estimate' to give it a try!
